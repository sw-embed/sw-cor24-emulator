Add a UART transaction log to the COR24 emulator. Track all UART input (bytes written to UART_DATA by host/user) and output (bytes read from UART_DATA by CPU) in chronological order. Handle overlapping input/output gracefully — if input and output alternate byte-by-byte (echo), coalesce them into logical input→output pairs. Add the log to the dump output (WasmCpu dump and any text-mode dump). Include a method to clear the log (e.g., on CPU reset). Write tests for the grouping/coalescing logic.
